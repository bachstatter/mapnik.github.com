# mapnik.org

Home for the Mapnik project on the web.

Uses [jekyll](https://github.com/mojombo/jekyll).

## Setup

### Install jekyll (depends on Ruby, rubygems and Ruby development packages)

    gem install jekyll nokogiri

### Run jekyll

    jekyll serve

### View the site at:

    http://127.0.0.1:4000/
